HTML Linguagem de Marcação de Hipertexto

Tutoriais para iniciantes

HTML (Linguagem de Marcação de Hipertexto) é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web.
Outras tecnologias além do HTML geralmente são usadas para descrever a aparência/apresentação (CSS) ou a funcionalidade/comportamento (JavaScript) de uma página da web.




Introdução

O que é HTML?


HTML é uma linguagem de marcação utilizada na construção de páginas na web. Documentos HTML podem ser interpretados por navegadores. A tecnologia é fruto da junção entre os padrões HyTime e SGML. HyTime é um padrão para a representação estruturada de hipermídia e conteúdo baseado em tempo.

O que é CSS?


Cascading Style Sheets (CSS) é usado para formatar o layout de uma página da web. Com CSS, você pode controlar a cor, a fonte, o tamanho do texto, o espaçamento entre os elementos, como os elementos são posicionados e dispostos, quais imagens de fundo ou cores de fundo devem ser usadas, exibições diferentes para dispositivos e tamanhos de tela diferentes e muito mais.

O que é JavaScript?


JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente com HTML e CSS, O JavaScript é uma das três principais tecnologias da World Wide Web.




Web e Front-end

A web, abreviação de World Wide Web, é um sistema de informação global que permite o acesso e a compartilhamento de documentos e recursos por meio da internet. Ela consiste em uma vasta rede de servidores interconectados que hospedam páginas da web, conteúdo multimídia, aplicativos e outros recursos acessíveis através de navegadores da web. A web utiliza protocolos padrão, como HTTP e HTTPS, para transferir dados entre os clientes (como navegadores) e os servidores, proporcionando uma experiência de navegação e interação na internet.

Protocolos Importantes


  1. HTTP - Hypertext Transfer Protocol
    2.

    É o protocolo principal da web responsável pela busca de dados solicitados pelo usuário em um servidor. O HTTPS (Hypertext Transfer Protocol Secure) é uma versão segura do HTTP que utiliza criptografia para proteger a comunicação entre o cliente e o servidor, garantindo a segurança e a privacidade dos dados transmitidos.


  2. TCP/IP - Transmission Control Protocol/Internet Protocol
    3.

    O TCP/IP é o conjunto de protocolos que permite a comunicação entre dispositivos em redes de computadores, incluindo a Internet. Ele define como os dados são transmitidos e roteados entre dispositivos em uma rede, garantindo uma conexão confiável e estável.


  3. DHCP - Dynamic Host Configuration Protocol
    4.

    O DHCP é um protocolo utilizado para atribuir endereços IP e outras configurações de rede automaticamente a dispositivos em uma rede local. Permite que dispositivos obtenham informações de configuração automaticamente ao se conectar à rede.


  4. FTP - File Transfer Protocol
    5.

    O FTP é um protocolo utilizado para transferir arquivos entre um cliente e um servidor em uma rede. Ele permite o upload e o download de arquivos de e para servidores remotos, sendo amplamente utilizado para transferência de arquivos em sites e servidores web.


  5. SMTP - Simple Mail Transfer Protocol
    6.

    O SMTP é o protocolo padrão para envio de e-mails pela Internet. Ele é utilizado para enviar mensagens de e-mail de um servidor de e-mail para outro. O SMTP define como os servidores de e-mail comunicam entre si para entregar mensagens aos destinatários corretos.

OUTROS COMANDOS IMPORTANTES


  • !DOCTYPE: Define o tipo de documento e a versão do HTML sendo usado.

    • Texto e Artigos

  • h1 a h6: Títulos de diferentes tamanhos, onde h1 é o maior e h6 é o menor.
  • p: Define um parágrafo no documento HTML.

    • Listas

  • ul e ol: Define listas não ordenadas e ordenadas, respectivamente.
  • li: Representa um item de lista dentro de ul ou ol.

    • Tabelas

  • table: Define uma tabela para organizar dados.
  • tr: Define uma linha dentro de uma tabela.
  • td: Define uma célula dentro de uma linha de tabela.
  • th: Define uma célula de cabeçalho dentro de uma linha de tabela.
